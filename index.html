01

Introduction

What is CSS?

CSS (Cascading Style Sheets) controls the visual presentation of HTML documents. The word "cascading" refers to the priority scheme used to determine which style rules apply when multiple rules target the same element.

CSS can be applied in three ways: inline (via the style attribute), internal (inside a <style> tag), and external (linked .css file — the recommended approach).

MethodSyntaxSpecificity
Inlinestyle="property: value"Highest (1000)
Internal<style> selector { } </style>Medium
External<link rel="stylesheet" href="file.css">Medium
CSS — Three Methods
/* 1. External (recommended) — in a .css file */
/* Linked via: <link rel="stylesheet" href="styles.css"> */

body {
  background: #0f0f0f;
  color: #e8e4df;
}

/* 2. Internal — inside <style> in HTML head */
h1 {
  font-size: 3rem;
  color: #e8a032;
}

/* 3. Inline — directly on element */
<p style="color: red;">Red text</p>

Margins

Outer Spacing

Margin creates space OUTSIDE the element's border. It is transparent and pushes neighboring elements away. Margins can collapse vertically (margin collapsing).

PropertyValuesNotes
margin1-4 values (TRBL)Shorthand for all sides
margin-top/right/bottom/leftpx, em, %, autoIndividual sides
margin: autoautoCenters block elements horizontally
margin: 0 auto0 autoCenter block with no vertical margin
CSS — Margins
/* All sides: 20px */
.m-all { margin: 20px; }

.m-vh { margin: 10px 30px; }

/* Top: 5px, Right: 10px, Bottom: 15px, Left: 20px */

/* Center horizontally */
.m-center {
  width: 200px;
  margin: 0 auto;

Box Model

The CSS Box Model

Every HTML element is a rectangular box consisting of four layers: Content → Padding → Border → Margin. The box-sizing property controls whether width/height include padding and border.

ValueWhat width includes
content-box (default)Only the content area
border-boxContent + Padding + Border
CSS — Box Model
/* Recommended global reset */
*, *::before, *::after {
  box-sizing: border-box;
}

/* With border-box: total width = 200px */
.box {
  box-sizing: border-box;
  width: 200px;
  padding: 20px;
  border: 5px solid #e8a032;
  /* Content area = 200 - 40 - 10 = 150px */
}

Media Queries

Responsive Design

Media queries apply CSS rules based on device characteristics (width, height, orientation, resolution, etc.). They are the foundation of responsive design.

FeatureExamplePurpose
max-width(max-width: 768px)Styles for screens up to 768px
min-width(min-width: 1024px)Styles for screens 1024px+
orientation(orientation: portrait)Portrait vs landscape
prefers-color-scheme(prefers-color-scheme: dark)Dark mode detection
prefers-reduced-motion(prefers-reduced-motion: reduce)Disable animations
CSS — Media Queries
/* Mobile first approach */
.container {
  padding: 16px;
}

@media (min-width: 768px) {
  .container { padding: 32px; max-width: 720px; }
}
@media (min-width: 1024px) {
  .container { max-width: 960px; }
}

/* Dark mode */
@media (prefers-color-scheme: dark) {
  body { background: #0f0f0f; color: #e8e4df; }
}

/* Respect reduced motion */
@media (prefers-reduced-motion: reduce) {
  * { animation: none !important; transition: none !important; }
}

Pseudo-classes select elements based on their state, not their position in the DOM. They start with a single colon :.

Pseudo-classPurpose

:disabled / :enabledForm element state
:emptyElement has no children
